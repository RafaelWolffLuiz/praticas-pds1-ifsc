Em programação, o que é uma biblioteca?

1 - É um termo utilizado para definir o código fonte compilado, qual foi produzido por terceiros
 que assim torna possível adicionar novas funções ao código fonte do software

2 - O que é a biblioteca Swing do Java e qual é o seu propósito?
Swing é uma biblioteca do Java para criar interfaces gráficas.

3 - Quais são os componentes de interface gráfica de usuário (GUI) fornecidos pela biblioteca Swing para:
Botão     JButton                   
Campo de Texto       JTextField        
Caixa de Seleção    JCheckBox       
Barra de Progresso    JProgressBar   
Botão de Opção     JRadioButton 
Caixa de Combinação  JComboBox

4 - Quais são os propósitos dos layouts no Swing? E quais são os layouts possíveis?
 Os propósitos dos layouts de swing são feitos para a organização e criação da interface , e os layouts possíveis são variados como: Botão (JButton), Campo de Texto (JTextField), caixa de seleção (JCheckBox) e etc. 

5 - Como é possível fazer a validação de entrada de dados em campos de texto?
Através do JOptionPane que fornece acesso a métodos de entrada/leitura de dados em uma pequena janela

6 - Pesquise e descreva três principais características entre a biblioteca Swing, a biblioteca AWT (Abstract Window Toolkit) e o JavaFX?
Swing: O Swing é uma biblioteca gráfica, qual fornece componentes de interface do usuário muito bons. Além de ter temas personalizados e recursos de internacionalização, possui opções avançadas de desenho e manipulação de imagens.
AWT: A biblioteca AWT é um versão anterior da Swing e por isso possui menos recursos, sendo eles: Um conjunto básico na criação das interfaces gráficas do usuário. E os aplicativos podem possuir diferentes aparências dependendo do sistema operacional.
JAVAFX: Usa a Linguagem de marcação XML para determinar a interface do usuário, assim se tornando mais fácil e rápido o desenvolvimento. Assim como AWT e Swing, detém um conjunto de componentes como botões, menus, tabelas e gráficos. É muito rica em recursos e com facilidade de uso na criação dos aplicativos de interface do usuário.
       Ambas as 3, fornecem um conjunto de componentes de UI para criar interfaces gráficas de usuário em Java.
